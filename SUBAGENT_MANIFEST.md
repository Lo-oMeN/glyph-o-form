## Delegation Rules

**I spawn subagents when:**
1. Task is parallelizable (research vs. code vs. design)
2. Task exceeds 30 minutes of focused work
3. New domain expertise needed (I don't know → research)
4. Testing required (I build → they validate)

**I integrate results when:**
1. Subagent completes with actionable output
2. Research reveals new opportunity
3. Test results require iteration
4. Multiple threads converge on single insight

**I escalate to you when:**
1. Strategic decision required (which axis to pursue)
2. Technical blocker I can't resolve
3. Resource constraint (need API key, hardware, etc.)
4. Ethical/design choice (what should the glyphs feel like?)
